# Knowledge System

## Key Files
- **Knowledge Base Rules:** `01_library/KNOWLEDGE_BASE_RULES.md` — wiki contract
- **Master Catalog:** `01_library/index.md` — updated on every /ingest
- **Event Log:** `log.md` — append-only chronology

## Page Types
- **Source note** — 1 source = 1 file (articles/, podcasts/, books/)
- **Topic page** — 1 topic = compiled knowledge from many sources (topics/)

## Operations
- **Ingest:** raw/ + source note + topic page check + index update + log
- **Query -> Save:** good answers become concept/topic pages
- **Lint:** orphans, broken links, missing backlinks, tag validation
