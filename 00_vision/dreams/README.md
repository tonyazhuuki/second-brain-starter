## Мечты: методика (RAW → Structured)

**Зачем:** удерживать живой «поток мечт» и переводить его в ясные векторы и цели, не теряя энергию.

### RAW (свободный поток)
- Пиши быстро, без редактуры и «как надо»
- Фиксируй картинки/сцены/ощущения
- Сортировка не нужна — важна честность и плотность сигнала

**Используй:** `dreams_guidance.md` как заготовку.

### Structured (навигация и связь с целями)
- Разложи мечты по сферам (здоровье, отношения, творчество, работа, капитал…)
- Для каждого пункта задай «ближайший честный масштаб» и ссылку в цели (год/месяц)

**Используй:** `structured_template.md` как заготовку.

### Связь с целями
- Год: `../goals/2026.md`
- Месяц: `../../04_therapy/sessions/ГОД/` (копируй `_month_template.md` → `YYYY_MM.md`)

> Контракты «мечта→цель» можно оформлять прямо в файлах целей; отдельная папка не требуется.
