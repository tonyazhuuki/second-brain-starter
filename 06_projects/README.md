## 06_projects — ваши проекты

Стартовая структура без личных проектов. Создайте свои папки на основе заготовок:

- `your_project_1/` — пример папки проекта (переименуйте)
- `your_project_2/` — ещё один слот

Рекомендуемая структура проекта:
```
your_project_X/
  README.md        # краткая формулировка, цели, метрики
  roadmap.md       # вехи, версии, релизы
  research.md      # идеи/гипотезы/инсайты
  notes/           # рабочие заметки
  assets/          # файлы/скриншоты/схемы
```
