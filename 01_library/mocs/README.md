# MOCs — Maps of Content

Эта папка хранит карты содержания раздела `01_library` (обзоры/сводки по книгам, подкастам, практикам, исследованиям).

Рекомендации:
- Создавай MOC‑файлы по мере роста контента, например: `books.md`, `podcasts.md`, `practices.md`, `research.md`.
- Следуй правилам ссылок и бэклинков: см. `../../.cursor/rules/linking_policy.mdc`.
- MOC — индекс → элементы; обратные бэклинки из элементов добавляй только если это повышает навигацию.
