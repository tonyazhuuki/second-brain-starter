# 📚 Books — как вести раздел

## Назначение
Собираем книги с ценными идеями и применением. Карточки — в `canon.md`, подробные разборы — отдельными файлами.

## Канон (index)
- `canon.md` — список карточек книг (когда/почему/суть/изменения/перечитывать/глубже).
- Если есть подробный разбор — добавь ссылку на файл. Если нет — оставь «будет добавлено».

## Ссылки
- Следуй стандарту ссылок и бэклинков: см. `../../.cursor/rules/linking_policy.mdc`.
- Проверка: `make links-validate`.
