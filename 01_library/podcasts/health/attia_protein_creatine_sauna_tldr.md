# 🍗 Протеин, креатин и сауна — что реально работает (Attia x Rhonda Patrick)

## 🧠 ТЕЗИС
RDA по белку занижен; прагматичный минимум для взрослых — ~1.2 г/кг/день, у старших — выше. Креатин безопасно поддерживает силу/когнитивные функции. Сауна снижает кардиориск при дозированной тепловой нагрузке.

## 🔑 КЛЮЧЕВЫЕ ИДЕИ
- RDA 0.8 г/кг — «минимально допустимый», а не оптимум; современные **изотопные** методы показывают недооценку потребности.
- **Минимум для «баланса»**: ≈1.2 г/кг/день; у старших и при тренировках — 1.2–1.6 г/кг.
- **Анаболическая резистентность** чаще из‑за неактивности, чем из‑за возраста → больше стимулов (сила+доза белка за приём).
- Критично распределение: **каждый приём 25–40 г** с достаточной лейцином (~2–3 г).
- **Креатин моногидрат**: 3–5 г/день длительно — прирост силы/мощности; есть данные по когнитивной поддержке и у вегетарианцев.
- **Сауна**: регулярные сессии (например, 80–90°C, 2–4×/нед) связаны с ↓ кардиориска; следить за гидратацией/давлением.
- У женщин старшего возраста потребление **≈1.2 г/кг/день** связано с **~30%** меньшим риском frailty (упомянуто в интервью).

## 🗺️ СТРУКТУРА/СЕГМЕНТЫ
- Почему RDA по белку занижен (история **nitrogen balance** vs **stable isotope**)  
- Минимум vs оптимум, распределение белка по приёмам  
- Анаболическая резистентность: роль неактивности  
- Креатин: доза, безопасность, кому особенно полезен  
- Сауна: частота/температура, предосторожности

## ⚖️ АРГУМЕНТЫ ↔ ВОЗРАЖЕНИЯ
- За повышение белка: изотопные исследования показывают недобор при 0.8 г/кг → потери мышц с возрастом.  
- Возражение «много белка вредно»: у здоровых почек дозы до 1.6–2.2 г/кг в исследованиях не ухудшали функцию; при ХБП — индивидуально с врачом.

## 📊 ДАННЫЕ/ЦИФРЫ
- Минимум: **~1.2 г/кг/день** (≈ +50% к RDA).  
- Приём: **25–40 г белка** с лейцином 2–3 г.  
- Креатин: **3–5 г/день** (без «загрузки»).  
- Сауна: **2–4×/нед**, 15–25 мин при 80–90°C (адаптировать под самочувствие).
- Женщины 60+: **1.2 г/кг/день → ~30%** ниже риск frailty.

## 📚 ГЛОССАРИЙ
- RDA — рекомендованная суточная норма (минимум, не оптимум).
- Stable isotope — «золотой стандарт» оценки белкового обмена (меченый фенилаланин) vs устаревший nitrogen balance.
- Анаболическая резистентность — сниженный ответ мышцы на аминокислоты/нагрузку.

## 💬 ЦИТАТЫ
- «RDA — это **минимум**, а не оптимум».  
- «У неактивных людей **анаболическая резистентность** — главный враг мышц».

## 🧩 НЕЯСНО
- Точные пороги белка для разных групп (женщины 40+, вегетарианцы) требуют персонализации.
