# topics/

Topic pages compile knowledge from multiple source notes into one place.

**When to create:** 3+ source notes on the same theme.
**Format:** compiled conclusions + data + links to all source notes.

Example: after ingesting 5 podcasts about protein, create `topics/protein.md` that combines all knowledge into one document.

See [KNOWLEDGE_BASE_RULES.md](../KNOWLEDGE_BASE_RULES.md) for full specification.
